### Задание 3

3.1 С помощью этой команды, мы ищем на какую позицию нам нужно восстановиться: 
mysqlbinlog --start-datetime="2020-03-11 20:05:00" \
                   --stop-datetime="2020-03-11 20:08:00" --verbose \
         /var/lib/mysql/bin.123456 | grep -C 15 "DROP TABLE"
         
С помощью этой комманды мы восстанавливаемся на выбраную позицию:
 mysqlbinlog --start-position=155 --stop-position=232 /var/lib/mysql/bin.123456 \
         | mysql -u root -p

3.2 Ни в одном из случаев репликация не будет давать преимущества перед использования бэкапов, так как репликация не заменяет резервное копирование, а дополняет его. Только при использовании этих двух подходов можно одновременно можно сгладить минусы и использовать плюсы одного и другого подходов и получить реальное преимущество.
